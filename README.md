# Futbol.AI
# ⚽ Futbol.AI  

Bienvenido a **Futbol.AI**, el punto en el que la **inteligencia artificial 🤖** y el **fútbol ⚽** se juntan para una gran experiencia tecnológica.

---

## 🌍 Rutas Públicas

### 🗓️ Programación de Partidos  
Simulación del calendario de partidos: horarios locales, equipos favoritos y datos obtenidos desde la base de datos.

### 📰 Noticias  
Las últimas novedades del fútbol, con publicaciones creadas por usuarios administradores y visibles para toda la red.

### 📊 Tabla de Posiciones  
Consulta de los puntajes de los equipos de las ligas más importantes de Europa, con información extraída directamente desde Sports-API.

---

## 🔐 Rutas Privadas  

Tras iniciar sesión, aparecen las rutas protegidas:

### 🔮 Oráculo  
Un modelo de inteligencia artificial creado con una red densa y un dataset de partidos (2000–2025). Predice resultados basándose en datos de la base de datos.  
Además, funciona como **chat inteligente** para consultas futbolísticas naturales.

### 🧑‍🤝‍🧑 Centro Social  
Un centro social con seguimiento de usuarios, contenido multimedia, búsquedas, amistad, entre otras funciones.

### 💬 Tablero  
Publicación de mensajes y opiniones en un chat global accesible por toda la comunidad de FutbolAI.

## 🛠️ Funciones de Administrador  

Disponibles únicamente para el usuario administrador designado.

### 📝 Administrar Noticias  
Crear, editar y manejar publicaciones con imágenes y contenido textual. Al finalizar, aparecen como públicas.

### 👤 Administrar Usuarios  
Crear usuarios, editar, eliminar y gestionar accesos dentro del sistema.

### 🏆 Administrar Equipos  
Crear equipos para la base de datos, inicializando todas sus métricas por defecto en **0**.

### 🏟️ Administrar Partidos  
Agregar, modificar o eliminar partidos programados o realizados, visibles para las demás pantallas.

### 📂 Archivos del Servidor  
Subir o descargar archivos desde la carpeta **/Files**, con iconos según el tipo de archivo.
